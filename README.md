# Tata Steel Crack Prediction

This project is based on crack detection in steel surfaces using Deep Learning and CNN (Convolutional Neural Network). The model is used to identify defects from steel images and helps in improving quality inspection in industries.

The project contains a Jupyter Notebook for model building and a Streamlit application for testing predictions through a simple interface.

---

## Files Included

```text
streamlit_app.py
Tata steel Crack Prediction.ipynb
```

## Technologies Used

- CNN

- OpenCV

- Matplotlib

---

## How to Run

Install the required libraries:

```bash
pip install streamlit tensorflow opencv-python numpy matplotlib
```

Run the Streamlit app:

```bash
streamlit run streamlit_app.py
```
